# Soniq

Soniq is an AI-powered voice synthesis and voice conversion project that generates natural-sounding speech while preserving speaker characteristics.

## Features
- Voice Conversion
- Voice Synthesis
- Speaker Identity Preservation
- High-Quality Audio Generation
- Audio Enhancement Pipeline

## Technologies Used
- Python
- PyTorch
- Seed-VC
- BigVGAN
- WORLD Vocoder

## How to Run

```bash
pip install -r requirements.txt
python synthesiser.py
